# Лендинг-стартер для воркшопа — Спецификация

## Executive Summary

Одностраничный лендинг для привлечения занятых предпринимателей на бесплатный воркшоп «Слепой импорт — увидеть реальность» (28-29 марта 2026). Альтернатива длинному лонгриду для аудитории, которая не читает. Цель: 400 регистраций за 7 дней через Telegram-бот.

## Problem Statement

**Текущая ситуация:**
- Лонгрид конвертит <5% в регистрацию
- Часть аудитории (занятые предприниматели) не читает длинный текст
- Нужен короткий «крючок» для быстрого принятия решения

**Почему сейчас:**
- Воркшоп 28-29 марта — осталось 8 дней
- Цель: 400 регистраций (для достижения 2 млн выручки)
- Трафик уже идёт: Яндекс.Директ, Авито, TG-канал

## Success Criteria

| Метрика | Цель |
|---------|------|
| Регистрации в боте | 400 за 7 дней |
| Конверсия визит→бот | >10% (vs <5% лонгрида) |
| Среднее время на странице | <90 секунд (быстрое решение) |
| Bounce rate | <60% |

## User Personas

**Основной:** Занятый предприниматель, импортирует из Китая
- Нет времени читать — нужна суть за 2 минуты
- Уже работает через карго/посредников
- Чувствует риски, но не понимает где
- Смотрит с телефона между делами

**Вторичный:** Скептик / холодная аудитория
- Ещё не доверяет, нужен быстрый «крючок»
- Хочет понять «что это мне даст» за 30 секунд
- Если зацепило — готов перейти к лонгриду

## User Journey

```
[Трафик: Директ/Авито/TG]
    ↓
[Лендинг: Hero — цепляет за 5 сек]
    ↓
[Скролл: Боль → Решение → Эксперт → Программа → Бонусы]
    ↓
[Выбор:]
├── [Кнопка «Зарегистрироваться»] → t.me/AShumilova_bot?start=landing
└── [Ссылка «Подробнее»] → лонгрид (для тех, кто хочет прочитать)
```

### СЕКЦИЯ 2 — БОЛЬ (проблема)

**Приоритет:** P0

**Фон:** Белый

**Заголовок H2:**
> Три ситуации — одна причина

**3 карточки историй (из лонгрида, сокращённые):**

**Карточка 1: «Прямая фабрика» оказалась перекупщиком**
> Импортёр годами был уверен, что работает с производителем. На самом деле — через посредника. Переплачивал и не знал.

**Карточка 2: «Белый ввоз» оказался карго с документами**
> Выбрал компанию с самой низкой ценой. Документы были. Схема развалилась при первом запросе из налоговой.

**Карточка 3: Документы не сошлись**
> Поставщик сам оформил экспорт. Наименования, вес, цена — ничего не совпало с импортом. Штрафы и доначисления — импортёру.

**Вывод (выделен):**
> Человек платил, отвечал головой — но не контролировал процесс.

**Acceptance Criteria:**
- [ ] Каждая история <50 слов
- [ ] Визуальное отличие карточек (иконки или цвет)

## Technical Architecture

### Stack
- **Frontend:** Чистый HTML + CSS (как лонгрид)
- **Хостинг:** Vercel (новый репозиторий)
- **Аналитика:** Яндекс.Метрика (ID: 102625647)

### Data Flow
```
[Лендинг]
    ↓ клик по CTA
[Telegram Deep Link: t.me/AShumilova_bot?start=landing]
    ↓
[Бот регистрирует, выдаёт бонусы]
```

### Integrations
- **Telegram Bot:** @AShumilova_bot — deep link с UTM `?start=landing`
- **Лонгрид:** alexandra-ved-longread.vercel.app — ссылка «Подробнее»
- **Яндекс.Метрика:** trackLinks:true для отслеживания кликов по боту

## Design System

### Цвета (наследуем от лонгрида)
| Роль | HEX |
|------|-----|
| Акцент | #597dee |
| Текст | #1c1c1c |
| Фон светлый | #ffffff |
| Фон серый | #f5f5f5 |
| Фон тёмный | #111111 |
| Текст на тёмном | #ffffff |

### Типографика
- **Шрифт:** Open Sans (как лонгрид)
- **H1:** 40px (mobile: 28px)
- **H2:** 32px (mobile: 24px)
- **Body:** 18px (mobile: 16px)
- **Line-height:** 1.65

### Кнопки
- Border-radius: 12px
- Padding: 18px 36px
- Background: linear-gradient (золотой, как в лонгриде) или #597dee
- Hover: shadow + translateY(-2px)

### Адаптивность
- **Desktop:** max-width 860px (как лонгрид)
- **Mobile:** 100% ширина, увеличенные отступы, sticky CTA внизу

## Non-Functional Requirements

- **Performance:** Lighthouse >90
- **Mobile-first:** Оптимизировано для мобильных (основной трафик)
- **SEO:** meta-теги, Open Graph для шеринга
- **Accessibility:** Контрастность AAA, alt для картинок

- Видео на лендинге
- Анимации (кроме hover)
